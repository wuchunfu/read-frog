## 👋🏻 Начало работы и сообщество

Цель Read Frog — дать изучающим языки удобный, интеллектуальный и персонализированный опыт обучения. В эпоху ИИ это стало возможным, но на рынке все еще мало продуктов, которые действительно закрывают эту потребность. Поэтому мы решили создать такой инструмент сами.

Если вы пользователь или разработчик, Read Frog может стать важной частью этого пути. Проект активно развивается, и мы будем рады сообщениям о любых [проблемах][issues-link].

## ✨ Возможности

Read Frog превращает обычное чтение в интернете в иммерсивный процесс изучения языка.

### 🔄 [Двуязычный режим / Только перевод][docs-tutorial]

Переключайтесь между двумя режимами отображения перевода. **Двуязычный режим** показывает исходный текст рядом с переводом, что удобно для обучения и сравнения. Режим **только перевод** заменяет исходный текст и делает чтение чище.

Если перевод уже включен, расширение автоматически переведет видимый контент заново при смене режима, без перезагрузки страницы.

### ✨ [Перевод выделенного текста][docs-tutorial]

Выделите любой текст на странице, чтобы открыть умную панель инструментов. **Перевод** выводит результат в реальном времени, **объяснение** дает подробный разбор под ваш уровень, а **озвучивание** читает текст вслух через TTS.

Панель автоматически остается в пределах окна, поддерживает перетаскивание и работает на любых сайтах.

### 🧠 [Перевод с учетом контекста][docs-tutorial]

Позвольте ИИ понимать полный контекст читаемой страницы. Read Frog извлекает заголовок и краткую Markdown-версию содержимого, чтобы перевод был точнее и лучше соответствовал ситуации.

Технические термины переводятся в рамках предметной области, литературные выражения сохраняют оттенки смысла, а неоднозначные фразы трактуются по окружающему тексту.

### 🎬 [Перевод субтитров][docs-tutorial]

Переводите субтитры YouTube прямо в видеоплеере. Смотрите контент на иностранных языках с переводом рядом с оригинальными субтитрами.

### 🔊 [Озвучивание текста (TTS)][docs-tutorial]

Слушайте выделенный текст с качественными AI-голосами. **Edge TTS** полностью бесплатен и предлагает 150+ голосов для 80+ языков, включая китайский, английский, японский, корейский и многие другие. Можно настроить скорость, тон и громкость.

Автоопределение языка и сопоставление голосов помогают выбрать подходящий голос. Длинный текст делится по естественным границам предложений, а следующий фрагмент предварительно загружается.

### 📦 [Пакетные запросы][docs-tutorial]

Снижайте расходы на API до 70% благодаря интеллектуальному объединению запросов. Read Frog группирует несколько переводов в один API-вызов, уменьшая накладные расходы и использование token без потери качества.

Система включает повторы с экспоненциальной задержкой и автоматический fallback к одиночным запросам, если пакетная обработка не удалась.

### 🤖 [20+ AI-провайдеров][docs-tutorial]

Подключайтесь к 20+ AI-провайдерам через Vercel AI SDK: OpenAI, DeepSeek, Anthropic Claude, Google Gemini, xAI Grok, Groq, Mistral, Ollama и другим. Для каждого можно настроить endpoint, API key и модель.

Также доступны бесплатные варианты: Google Translate, Microsoft Translate и DeepLX для базового перевода без затрат.
