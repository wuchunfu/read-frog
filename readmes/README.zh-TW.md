Read Frog 的願景，是為各種程度的語言學習者提供易用、智慧且個人化的學習體驗。在 AI 時代，這件事已經變得可行，但市場上仍很少有產品真正滿足這個需求。因此我們決定自己動手，讓世界最終不再依賴人類語言教師。

無論你是使用者還是開發者，Read Frog 都會是你走向這個願景的重要工具。Read Frog 目前仍在積極開發中，歡迎針對遇到的任何[問題][issues-link]提供回饋。

## ✨ 功能

透過 Read Frog 的強大功能，把日常網頁閱讀變成沉浸式語言學習旅程。

### 🔄 [雙語 / 僅譯文][docs-tutorial]

在兩種翻譯顯示模式間順暢切換。**雙語模式**會同時顯示原文與譯文，適合學習與對照；**僅譯文模式**會直接替換原文，提供更清爽的閱讀體驗。

翻譯啟用時切換模式，擴充功能會自動重新翻譯所有可見內容，讓過渡更平順且不需要重新整理頁面。

### ✨ [劃詞翻譯][docs-tutorial]

在網頁上選取任何文字，即可顯示智慧工具列。**翻譯**會即時串流輸出結果，**解釋**會依照你的語言程度提供詳細說明，**朗讀**則可用文字轉語音唸出內容。

工具列會自動定位在視窗內，支援拖曳互動，並能在各種網站上運作，非常適合閱讀時快速查詢。

### 🧠 [上下文感知翻譯][docs-tutorial]

讓 AI 理解你正在閱讀內容的完整上下文。啟用後，Read Frog 會擷取頁面標題與精簡的 Markdown 頁面內容，交給 AI 產生更準確、更貼合語境的翻譯。

技術術語會依所屬領域正確翻譯，文學表達能保留細膩語感，歧義片語也會依上下文而不是孤立文字來解讀。

### 🎬 [字幕翻譯][docs-tutorial]

直接在影片播放器中翻譯 YouTube 字幕。觀看外語內容時，譯文會與原始字幕一起顯示，讓影片也能成為語言學習素材。

### 🔊 [文字轉語音 (TTS)][docs-tutorial]

使用高品質 AI 語音朗讀任何選取文字。由 **Edge TTS** 驅動，完全免費，提供 150+ 種語音並支援 80+ 種語言，包括中文、英文、日文、韓文等。你可以自由調整語速、音調與音量。

自動語言偵測與依語言對應語音，能為每種語言選擇最合適的聲音。智慧句子切分能自然處理長文本並預先載入下一段，適合發音練習與聽力學習。

### 📦 [批次請求][docs-tutorial]

透過智慧批次處理，最多可節省 70% API 成本。Read Frog 會把多個翻譯請求合併為一次 API 呼叫，在維持品質的同時降低開銷與 token 使用量。

系統包含指數退避重試機制，並在批次失敗時自動退回單次請求，所有流程都會在背景透明處理。

### 🤖 [20+ AI 服務商][docs-tutorial]

透過 Vercel AI SDK 連接 20+ AI 服務商：OpenAI、DeepSeek、Anthropic Claude、Google Gemini、xAI Grok、Groq、Mistral、Ollama 等。你可以為每個服務商設定自訂端點、API 金鑰與模型。

此外也提供免費翻譯選項：Google 翻譯、Microsoft 翻譯與 DeepLX，適合零成本的基礎翻譯。
